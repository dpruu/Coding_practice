# 9. 수학 Ⅱ

> 소수 찾기

```python
N = int(input())
nums = map(int, input().split())
ans = 0
for num in nums:
    S = 0
    if num > 1:
       for i in range(2,num):
        if num % i == 0:
          S += 1
          break
       if S == 0:
         ans += 1
print(ans)
```

> 소수

```python
A = int(input())
B = int(input())
ans = []
for i in range(A,B+1):
    chk = 0
    if i > 1: # A가 1일 가능성이 있기 때문에 고려
        for j in range(2,i):
            if i % j == 0:
                chk += 1
                break
        if chk == 0:
            ans.append(i)

if len(ans) > 0:
    print(sum(ans))
    print(min(ans))
else:
    print(-1)
```
